# Play-by-Play Enhancements Design

> **Date:** 2026-03-29
> **Status:** Approved
> **Scope:** Frontend-only (no backend/real-time infrastructure changes)

---

## Layout

**Expanded 4-Zone Layout** — existing 3-column grid (`200px | 1fr | 260px`) plus a new bottom ticker bar spanning full width.

```
┌────────────────┬──────────────────────────────┬─────────────────┐
│ Around the     │ Scoreboard (with drive stats) │ Box Score       │
│ League scores  │ Momentum bar                  │ (live-updating) │
│                │ Field (with RZ overlay)        │                 │
│ H2H Rivalry    │ Commentary (with penalty info) │                 │
│ (if human opp) │ Controls                       │                 │
├────────────────┴──────────────────────────────┴─────────────────┤
│ Bottom Ticker: Around-the-League Alerts                          │
└──────────────────────────────────────────────────────────────────┘
```

---

## 1. Drive Summary Strip

**Location:** Embedded in the scoreboard row (center panel), between team scores.

**Data:**
- Play count, total yards, elapsed time for the current drive
- Resets on possession change (turnover, punt, score, turnover on downs)
- Computed client-side by walking events backward from current `idx` to last possession change

**Display:** `Drive: 6 plays, 43 yds, 2:34`

---

## 2. Momentum Meter (Tug-of-War Bar)

**Location:** 6px bar between the scoreboard and field.

**Formula** (rolling window of last 8 plays):
| Event | Points |
|-------|--------|
| First down | +1 offense |
| Big play (20+ yds) | +2 offense |
| Touchdown | +3 offense |
| Turnover | -3 offense |
| Sack | +1 defense |
| 3-and-out (detected via down > 4) | +2 defense |

- Score clamped to [-5, +5], mapped to 0–100% bar position (50% = neutral)
- Bar fills with leading team's color gradient
- Glowing indicator dot at the current position
- Smooth CSS transition (0.5s) on position changes

---

## 3. Key Play Highlights

Enhance existing `bigPlay` system in FieldView:
- **Field border flash** — 1.5s colored border on TDs (gold), turnovers (red), big plays (orange)
- **Score pulse** — 0.3s scale-up animation on the score number when points are scored
- Triggered by existing `bigPlay` state + new CSS classes

---

## 4. Red Zone Intensity

When `yardLine >= 80` (inside opponent's 20):
- Translucent red overlay on the last 20% of the field green area
- Subtle "RED ZONE" text label in the overlay corner
- CSS transition (0.3s fade in/out)

---

## 5. Penalty Decision Popup (Hybrid Inline)

When `ev.penalty` exists on the current play:
- Commentary box shows play text + penalty detail below, separated by orange divider
- Format: `🚩 Defensive Holding — DECLINED (5 yds < 12 yd gain)`
- Commentary box border shifts to orange when penalty present
- No overlay, no layout disruption

---

## 6. OT Drama

When `quarter > 4`:
- Scoreboard shows pulsing "OVERTIME" badge (amber glow animation)
- Momentum bar gets an amber tint on the track
- Commentary templates get dramatic variants
- Field border gets subtle amber glow

---

## 7. Around-the-League System

### Bottom Ticker Bar
- Full-width bar below the 3-column grid
- Shows 2-3 items from other games: final scores, key events
- Generated by scanning all games in the same week
- Items rotate as user advances plays

### Overlay Toasts
- Pop up over the field area when other games had key moments
- **Timing model:** Map each game's events to 0–100% timeline. As user scrubs through their game, alerts fire at matching % points.
- Toast slides in from bottom, auto-dismisses after 3 seconds or on next play advance
- ~3-5 alerts per full game replay
- Shows: team abbreviation + event description + score

### Alert Generation
Scan all week games for:
- Touchdowns (especially go-ahead or tying)
- Turnovers (interceptions, fumbles)
- Close finishes (final score within 3 points)
- Overtime games

---

## 8. H2H Rivalry Stats

When focused game involves another human player's team (`ownerId` set):
- Small "RIVALRY" section in left panel below scores
- Series record (e.g., "PHI leads 3-1")
- Last matchup result
- Data source: completed games from `league.currentSeason.games` + `league.history` filtered by these two team IDs

---

## 9. Post-Game Highlights Reel

New section in GameRecapView (shown at "Final"):
- **Top 5 plays ranked by score swing** — absolute change in `homeScore - awayScore` differential
- Each entry: rank, play description, quarter, score change visualization
- **Clickable** — sets `idx` to that play's index, jumping the replay
- Requires an `onJumpToPlay?: (idx: number) => void` callback from GameCenterView

---

## 10. Live-Updating Box Score Enhancements

- **Points by quarter row** — Q1/Q2/Q3/Q4/OT breakdown, updates live as `idx` advances
- **Stat flash** — 0.5s amber highlight on changed numbers when stats update
- **TD badges** — small TD count badge next to player names with touchdowns
